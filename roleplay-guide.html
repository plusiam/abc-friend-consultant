내담자 역할

📌 내담자의 역할

친구 관계에서 겪은 어려움이나 고민을 이야기하는 사람입니다.

🎯 어떻게 준비하나요?

  • 실제 경험 활용: 친구와 있었던 갈등이나 속상했던 일 선택
  • 가상 상황 선택: 실제 경험이 어렵다면 예시 활용
    • 친한 친구가 나의 비밀을 다른 사람에게 말했을 때
    • 같이 놀던 친구들이 나를 빼고 놀 때
    • 친구가 나에게만 차갑게 대할 때
    • 그룹 과제에서 친구가 맡은 부분을 하지 않았을 때

💡 내담자 역할 팁

  • 상황을 구체적으로 설명하세요 (언제, 어디서, 누구와)
  • 그때의 감정을 솔직하게 표현하세요
  • 상담가의 질문에 성실히 답해주세요
  • 도움을 받고 싶은 마음으로 대화에 참여하세요

✅ 내담자 체크리스트

  • 이야기할 상황을 정했나요?
  • 언제, 어디서, 누구와 있었던 일인지 기억하나요?
  • 그때의 감정을 설명할 수 있나요?
  • 상담을 통해 어떤 도움을 받고 싶나요?

상담가 역할

📌 상담가의 역할

내담자의 이야기를 듣고 ABC 모델로 문제 해결을 돕는 사람입니다.

🎯 ABC 상담 5단계 (총 15분)

1단계: 따뜻한 시작 (2분)

  • 눈 마주치며 인사
  • 편안한 분위기 조성
"네 이야기를 듣고 함께 해결해보자"

2단계: A-상황 파악 (5분)

  • 육하원칙으로 질문
  • 구체적 상황 확인
"언제, 어디서 그런 일이 있었어?"

3단계: B-생각 탐색 (3분)

  • 그때의 생각 묻기
  • 판단 없이 경청
"그때 어떤 생각이 들었어?"

4단계: C-감정 이해 (2분)

  • 감정 확인·공감
  • 감정 강도 확인
"정말 속상했겠다"

5단계: 새로운 B와 C 찾기 (3분)

  • 다른 관점 제시: "혹시 다르게 생각해볼 수도 있을까?"
  • 긍정적 생각 찾기: "이 상황에서 배울 점은?"

💡 공감 표현 모음

"그런 일이 있었구나" / "네 마음이 이해돼" / "나라도 그랬을 거야" / "많이 속상했겠다"

⚠️ 피해야 할 것: 판단하기, 비난하기, 해결책 강요하기

기록자 역할

📌 기록자의 역할

상담 과정을 관찰하고 ABC 모델에 맞게 정리하는 사람입니다.

🎯 기록 방법

  • 조용히 관찰: 대화 방해하지 않고 집중해서 듣기
  • 핵심 포착: 중요한 내용을 메모하기
  • ABC 분류: 들은 내용을 A, B, C로 구분하여 정리
  • 웹 기록: ABC 친구상담소 웹사이트에 입력하기

✅ 기록자 체크리스트

📝 기록 예시

A: 점심시간에 친구들이 나를 빼고 밥을 먹었다.
B: 친구들이 나를 싫어하나보다.
C: 슬프고 외로웠다.
새 B: 친구들이 깜빡했을 수도 있다.
새 C: 조금 섭섭하지만 이해할 수 있다.

💡 기록자 팁

  • 대화를 요약해서 기록하세요
  • 감정 표현에 특히 주목하세요
  • 개인정보는 쓰지 마세요 (이니셜 사용)

📋 전체 진행 순서 (총 25분)

1️⃣ 시작 전 준비 (5분)

  • 역할 정하기 (가위바위보 또는 자원)
  • 자리 배치: 내담자↔상담가 마주보기, 기록자는 옆에서
  • 각자 역할 가이드 읽기
  • ABC 친구상담소 웹사이트 준비 (기록자)

2️⃣ 상담 진행 (15분)

전반부 (10분)
  • 시작 인사 (2분)
  • A-상황 파악 (5분)
  • B-생각 탐색 (3분)
후반부 (5분)
  • C-감정 이해 (2분)
  • 새로운 B, C 찾기 (3분)

3️⃣ 마무리 (5분)

  • 기록자가 정리한 내용 확인
  • 웹사이트에 최종 입력
  • 서로 역할 수행 칭찬하기
  • 느낀 점 나누기
🌟 성공적인 역할놀이를 위한 약속
  • 친구의 이야기를 비밀로 지켜주세요
  • 진지하게 참여해주세요
  • 서로를 존중하는 말을 사용해주세요
  • 완벽하지 않아도 괜찮아요. 연습이니까요!

📌 역할 교대

시간이 있다면 역할을 바꿔서 다시 해보세요. 모든 친구가 각 역할을 경험해보면 좋습니다!
